# RedSocial
Mi proyecto de BootStrap
Es una realidad que a día de hoy los frameworks son cada vez mas importantes y es por ello que empecé a estudiar Bootstrap.
Como trabajo final de este nos mandaron en clase realizar nuestra propia red social.

Facebook nunca me ha gustado. Lo digo públicamente, su interfaz no me convence. No puedo decir si es la paleta de colores, la intuitividad
o la razón pero no me ha gustado nunca su interfaz. Por ello decidí replicarlo pero en este caso a mi forma, a mi manera y es entonces cuando decidí 
usar una paleta de colores oscura. Un contraste entre el modo claro y oscuro. Ni vivo ni apagado. Elegante.
Recalcando siempre que es una opinión personal.
